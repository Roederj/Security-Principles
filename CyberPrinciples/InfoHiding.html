Default

When a variable is made public, it becomes readily accessible to anyone making changes to the program. Changes to critical variables can be made, compromising how the program works. In a security setting, variables containing sensitive information can be easily retrieved.

As a general rule, as little of a program should be interfaced with as possible. In this case, as little of the program should be made public as possible. Every accessible function is a point through which someone can interact and affect a program. If enough functions are available, it can even be possible to determine the exact inner workings of the program.
